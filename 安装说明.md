# 📲 安装说明 - 如何将APK安装到手机

## 🎯 安装步骤

### 方法1：USB连接安装（推荐）

#### 准备工作
1. 用USB数据线连接手机和电脑
2. 在手机上开启"USB调试"模式

#### 开启USB调试
**Android 手机通用步骤：**
1. 打开"设置"
2. 找到"关于手机"
3. 连续点击"版本号" 7次，开启"开发者选项"
4. 返回设置，进入"开发者选项"
5. 开启"USB调试"
6. 连接电脑时，手机会提示"允许USB调试"，点击"允许"

#### 安装命令
打开 PowerShell 或 CMD，执行：
```bash
cd "D:\我的文档\日常计划Flutter应用"
flutter install
```

或者直接使用 adb 命令：
```bash
adb install build\app\outputs\flutter-apk\app-release.apk
```

---

### 方法2：文件传输安装（最简单）

#### 步骤1：传输APK到手机

**方式A：微信/QQ传输**
1. 在电脑上打开微信/QQ
2. 找到APK文件：`D:\我的文档\日常计划Flutter应用\build\app\outputs\flutter-apk\app-release.apk`
3. 发送给"文件传输助手"或自己
4. 在手机上接收文件

**方式B：USB数据线传输**
1. 用USB线连接手机和电脑
2. 在电脑上打开"此电脑"，找到手机设备
3. 将APK文件复制到手机的"Download"文件夹

**方式C：云盘传输**
1. 上传APK到百度网盘/OneDrive等
2. 在手机上下载

#### 步骤2：在手机上安装

1. 在手机上找到APK文件
   - 打开"文件管理器"
   - 进入"Download"或"下载"文件夹
   - 找到 `app-release.apk`

2. 点击APK文件

3. 如果提示"禁止安装未知应用"：
   - 点击"设置"
   - 允许"安装未知应用"
   - 返回继续安装

4. 点击"安装"按钮

5. 等待安装完成（约5-10秒）

6. 点击"打开"开始使用

---

### 方法3：扫码安装

如果您有网站或云存储：
1. 将APK上传到网络位置
2. 生成下载链接或二维码
3. 手机扫码下载安装

## ⚙️ 允许安装未知应用

不同品牌手机的设置路径略有不同：

### 小米手机
1. 设置 → 隐私保护 → 特殊权限 → 安装未知应用
2. 选择"文件管理器"或"浏览器"
3. 允许安装

### 华为手机
1. 设置 → 安全 → 更多安全设置 → 安装外部来源应用
2. 选择"文件管理器"
3. 允许安装

### OPPO/vivo 手机
1. 设置 → 安全 → 未知来源
2. 允许安装未知应用

### 三星手机
1. 设置 → 生物识别和安全 → 安装未知应用
2. 选择"我的文件"
3. 允许安装

### 原生Android
1. 设置 → 应用和通知 → 高级 → 特殊应用权限
2. 安装未知应用
3. 选择"文件管理器"，允许安装

## ❓ 常见问题

### Q1: 提示"应用未安装"
**可能原因：**
- APK文件损坏 → 重新下载
- 手机存储空间不足 → 清理空间
- 系统版本过低 → 需要 Android 5.0+

### Q2: 安装后无法打开
**解决方案：**
1. 卸载后重新安装
2. 清除应用数据
3. 重启手机后再试

### Q3: 提示"解析包时出现问题"
**解决方案：**
- APK文件可能损坏，重新构建或下载
- 确认手机系统版本 ≥ Android 5.0

### Q4: 数据会丢失吗？
**不会！** 数据保存在云端（Supabase），即使：
- 卸载应用
- 换手机
- 重新安装

数据都会自动同步回来。

### Q5: 需要登录吗？
**不需要！** 这是个人使用的应用，无需注册登录。

### Q6: 占用多少空间？
- **APK大小**：约 20-30 MB
- **安装后**：约 40-50 MB
- **使用数据**：几乎不占空间（存云端）

### Q7: 耗电吗？
**不耗电！** 应用只在使用时运行，不会后台耗电。

## 📞 需要帮助？

如果安装过程中遇到问题：

1. **检查手机系统版本**
   - 设置 → 关于手机 → Android 版本
   - 需要 Android 5.0 或更高

2. **检查存储空间**
   - 至少需要 100MB 可用空间

3. **尝试重新下载APK**
   - 可能是文件传输过程中损坏

4. **重启手机**
   - 有时重启可以解决安装问题
